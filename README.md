### Installation

[Install Rust](https://www.rust-lang.org/tools/install)
```bash
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
```

### Using Cargo

- Create project:  
  `cargo new APP_NAME`
- build:  
  `cargo build`
- run:  
  `cargo run`  
  `cargo run -- ARGS HERE`
- test:  
  `cargo test`
